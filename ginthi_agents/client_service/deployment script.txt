Deployment Steps

First, create a virtual environment using the command:

python -m venv venv


To activate the virtual environment, run:

venv\Scripts\activate


Navigate to the ginthi_agents folder:

cd ginthi_agents


Run the following command to start the client server:

python -m client_service.main


To view the API documentation, open your browser and enter the link displayed in the terminal.
Example:

http://127.0.0.1:8005/docs
